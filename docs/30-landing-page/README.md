# Área Landing Page

## Objetivo

A **Landing Page** é a área pública de apresentação do produto, planos e fluxos de pré‑cadastro de novos clientes.

## Arquivos desta área

- `01-visao-geral.md` — visão geral da área de landing.
- `02-rotas.md` — visão geral das rotas públicas relacionadas à landing.
- `03-views.md` — organização básica das views Blade da landing.
