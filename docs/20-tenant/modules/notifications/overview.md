# Notifications — Overview

O módulo **Notifications** gerencia notificações internas para usuários do Tenant.

- Lista notificações recentes.
- Permite visualizar detalhes e marcar notificações como lidas.
- Expõe um endpoint JSON para consumo em frontends (ex.: badge de notificações).

> Informações extraídas de `NotificationController` e `routes/tenant.php`.
