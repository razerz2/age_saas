# Online Appointments — Overview

O módulo **Online Appointments** gerencia consultas cujo modo (`appointment_mode`) é "online".

- Lista apenas agendamentos online.
- Permite configurar instruções de acesso (link, app, instruções gerais e para o paciente).
- Permite o envio dessas instruções por email e WhatsApp.

> Informações extraídas de `OnlineAppointmentController` e `routes/tenant.php`.
