# Módulo: Forms

Mapa rápido do módulo de formulários personalizados do Tenant.

- `overview.md` — visão geral e fluxo de criação/edição.
- `routes.md` — rotas de gestão de formulários e builder.
- `views.md` — telas principais e partials de formulários.
- `backend.md` — controllers, requests e serviços.
- `frontend.md` — JS/CSS e padrões de UI ligados a forms.
- `database.md` — tabelas e relacionamentos relevantes.
- `permissions.md` — módulos/roles que controlam o acesso.
- `troubleshooting.md` — problemas comuns e checklist de diagnóstico.
