# Integrations — Views

Fontes: `IntegrationController`.

## Views principais

- `resources/views/tenant/integrations/index.blade.php`
  - Lista integrações paginadas.

- `resources/views/tenant/integrations/create.blade.php`
  - Formulário para criar uma nova integração.

- `resources/views/tenant/integrations/show.blade.php`
  - Detalhes de uma integração específica.

- `resources/views/tenant/integrations/edit.blade.php`
  - Formulário para editar uma integração existente.

## Layouts e componentes

- Layout base específico não foi inspecionado — provavelmente herda do layout Tenant padrão.
- Padrões globais de UI em `docs/00-global/03-padroes-frontend.md`.
