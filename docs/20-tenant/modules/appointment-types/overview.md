# Appointment Types — Overview

O módulo **Appointment Types** gerencia os tipos de atendimento (duração, preço, médico) no Tenant.

- Cada tipo de atendimento pertence a um médico.
- Controla duração, preço e atributos visuais (como cor).
- É utilizado pelo módulo de agendamentos para definir duração dos slots e apresentação das consultas.

> Informações extraídas de `AppointmentTypeController` e `routes/tenant.php`.
