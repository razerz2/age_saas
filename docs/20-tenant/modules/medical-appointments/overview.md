# Medical Appointments — Overview

O módulo **Medical Appointments** provê a sessão de atendimento médico do dia.

- Permite selecionar data e médicos (ou usar automaticamente o médico do usuário).
- Exibe lista de agendamentos do dia, filtrada por permissões do usuário.
- Permite atualizar o status dos atendimentos e navegar para o próximo agendamento.
- Permite visualizar detalhes do agendamento e, quando existir, respostas de formulários vinculados.

> Informações extraídas de `MedicalAppointmentController` e `routes/tenant.php`.
