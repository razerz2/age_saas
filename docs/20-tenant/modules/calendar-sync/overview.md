# Calendar Sync — Overview

O módulo **Calendar Sync** gerencia o estado de sincronização de agendamentos com integrações de calendário.

- Armazena informações de sincronização associadas a agendamentos específicos.
- Permite listar, criar, editar e remover estados de sincronização.

> Informações extraídas de `CalendarSyncStateController` e `routes/tenant.php`.
