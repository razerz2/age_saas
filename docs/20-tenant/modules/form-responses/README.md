# Módulo: Form Responses

Mapa rápido do módulo de respostas de formulários do Tenant.

- `overview.md` — visão geral e fluxo de respostas.
- `routes.md` — rotas de listagem, visualização e edição de respostas.
- `views.md` — telas principais e partials de responses.
- `backend.md` — controllers, requests e serviços auxiliares.
- `frontend.md` — JS/CSS associados quando existirem.
- `database.md` — tabelas e relacionamentos relevantes.
- `permissions.md` — módulos/roles que controlam o acesso.
- `troubleshooting.md` — problemas comuns e checklist de diagnóstico.
