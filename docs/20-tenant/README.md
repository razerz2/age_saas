# Área Tenant

## Objetivo

A **Tenant** é a área de trabalho de cada clínica, onde são gerenciados pacientes, médicos, calendários, agendamentos, formulários, integrações e relatórios específicos de cada tenant.

## Arquivos desta área

- `01-visao-geral.md` — visão geral e objetivos da área Tenant.
- `02-rotas.md` — visão geral das rotas principais da área.
- `03-estrutura-de-pastas.md` — onde vivem controllers, models, views e outros artefatos relacionados.
- `04-padroes-ui-tenant.md` — padrões de UI/UX específicos da área Tenant.
