# Área Platform

## Objetivo

A **Platform** é a área administrativa central do sistema, responsável por gerenciar tenants, planos, assinaturas, faturas, redes de clínicas e configurações globais.

## Arquivos desta área

- `01-visao-geral.md` — visão geral e objetivos da área Platform.
- `02-rotas.md` — visão geral das rotas principais da área.
- `03-estrutura-de-pastas.md` — onde vivem controllers, models, views e outros artefatos relacionados.
