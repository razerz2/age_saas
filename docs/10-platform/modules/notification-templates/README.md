# Modulo: Layouts de Email (Platform)

Este modulo documenta a gestao de **layouts de email** (header/footer/cores/logo) na Platform.

Notas de escopo:

- templates de email sao catalogos separados:
  - `modules/platform-email-templates/` (eventos SaaS/Platform)
  - `modules/tenant-email-templates/` (baseline email para eventos clinicos/Tenant)
- neste modulo nao existe CRUD de templates (apenas layouts).
