# Overview

Modulo de layout aplicado a emails do sistema.

Conceitos:

- **Template de email**: assunto + body (catalogos separados por escopo).
- **Layout de email**: header/footer/cores/logo (estrutura aplicada ao body no envio).

- `Tenant Email Templates`: `docs/10-platform/modules/tenant-email-templates/`
