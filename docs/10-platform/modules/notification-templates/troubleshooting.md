# Troubleshooting

## Preview nao reflete o layout ativo

- confirme qual layout esta marcado como `is_active`;
- ao ativar um layout, os demais sao desativados automaticamente no update.

## Logo nao aparece no preview

- se for upload, confirme `storage:link` e permissao do disk `public`;
- se for URL externa, confirme que a URL e acessivel pelo browser e que aponta para uma imagem valida.
