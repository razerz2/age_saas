# Frontend

UI (Platform):

- `index`: filtros por `name` e `enabled` + paginacao;
- `show`: exibicao de `key`, `display_name`, `subject`, `body` + botao `Testar envio`;
- `edit`: edicao de `subject`/`body` e toggle `ativo`, com acao `Restaurar Padrao`.

Modal `Testar envio`:

- pede `destination_email`;
- reabre automaticamente quando ha erro de validacao no campo.
